# 📝 Soal Latihan Praktek Coding – Conditional Statement di Dart

Buat program Dart yang meminta input umur seseorang, lalu tentukan kategori berdasarkan aturan:
- 0–12 → Anak-anak
- 13–17 → Remaja
- 18–59 → Dewasa
- ≥ 60 → Lansia

### Contoh Input/Output:
```
Masukkan umur: 15
Output: Anda termasuk kategori Remaja
 ```
